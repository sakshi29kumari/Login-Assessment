## ğŸ”„ Typical User Flow

### New User
1. Navigate to `/register`
2. Enter name, email, password
3. Click "Create Account"
4. Auto-redirected to profile
5. View user information
6. Click "Sign Out" to logout

### Returning User
1. Navigate to `/login`
2. Enter email and password
3. Click "Sign In"
4. Auto-redirected to profile
5. View user information
6. Click "Sign Out" to logout

### Protected Access
1. Try to access `/profile` without auth
2. Auto-redirected to `/login`
3. After login, can access profile
4. After logout, cannot access profile
